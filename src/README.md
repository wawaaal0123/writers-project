# Парсинг сайта «Электронная библиотека авторов Карелии»

## Описание

Скрипт для парсинга данных о писателях Карелии с сайта http://avtor.karelia.ru

## Что парсится

- ФИО писателя
- Даты рождения и смерти
- Место рождения
- Биография
- Список произведений (библиография автора)

## Как запустить

### 1. Установка зависимостей

bash
pip install requests beautifulsoup4

### 2. Запуск скрипта

python parser.py
Структура выходных данных
authors

id

full_name

birth_date

death_date

birth_place

biography

texts

id

title

type

author_id

year_written

### Примечание
Сайт использует кодировку windows-1251

Между запросами ставится пауза 1 секунда

Парсится указатель авторов по буквам
